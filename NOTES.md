## What Was Built

### 1. Task CRUD (fully functional)
- Model: `app/Models/Task.php` with `$fillable`, `$casts`
- Controller: `app/Http/Controllers/TaskController.php` (index, create, store, show, edit, update, destroy)
- Views: `resources/views/tasks/` → `index`, `create`, `edit`, `show`
- Routes: `Route::resource('tasks', TaskController::class)`
- Factory + Seeder: 10 fake tasks seeded

### 2. Client CRUD (fully functional)
- Model: `app/Models/Client.php` — fields: `nom`, `email`, `telephone`, `entreprise`
- Controller: `app/Http/Controllers/ClientController.php` with validation rules & custom messages
- Views: `resources/views/clients/` → `index` (with search), `create`, `edit`, `show`
- Routes: `Route::resource('clients', ClientController::class)`
- Factory + Seeder: 10 fake clients seeded
- Search scope: `Client::search($term)` searches nom, email, entreprise

### 3. Eloquent Avancé (Step 2)
**Task Accessors:**
- `$task->priority_label` → "Nulle" / "Basse" / ... / "Critique"
- `$task->priority_color` → Tailwind CSS color classes
- `$task->is_overdue` → true if past due and not completed

**Task Scopes:**
- `Task::pending()` — not completed
- `Task::completed()` — completed
- `Task::overdue()` — past due & not completed
- `Task::dueToday()` — due today
- `Task::byPriority($level)` — filter by priority

**Client Accessors:**
- `$client->initiales` → e.g. "JD" for "Jean Dupont"

**Client Scopes:**
- `Client::search($term)` — search by nom, email, entreprise

### 4. Blade Components (Step 3)
- `<x-form-input>` — reusable input with label, error display, hint
  - Used in client create/edit forms
  - Props: `name`, `label`, `type`, `value`, `placeholder`, `required`, `hint`

### 5. Authentication (Step 4) — Laravel Breeze
- Installed via `php artisan breeze:install blade`
- Routes protected with `middleware('auth')`
- Login / Register / Password Reset pages auto-generated by Breeze
- Navbar shows logged-in user initials + logout button
- Unauthenticated users are redirected to `/login`

### 6. Shared Layout (`resources/views/layouts/app.blade.php`)
- Sticky navbar with active link highlighting
- Context-aware CTA button (Nouvelle tâche vs Nouveau client)
- Flash success messages with green banner
- Auth-aware: shows user avatar + logout when logged in

## Run the Project

```bash
# Start PHP dev server
php artisan serve

# Start Vite (CSS/JS hot reload) — in a separate terminal
npm run dev

# Re-seed database (wipe + fresh)
php artisan migrate:fresh --seed
```
